# Portfolio Website

## Overview

This project is a personal portfolio website built using HTML, CSS, and Bootstrap 5. It showcases my skills, projects, and contact information while demonstrating responsive web design principles and accessibility best practices.

## Features

* Full-screen hero section with responsive navigation
* About Me section with skills badges
* Responsive projects grid
* Contact form with HTML validation
* Dark mode support using `prefers-color-scheme`
* Mobile-friendly design
* Semantic HTML structure

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

## Credits

Bootstrap 5 CSS framework was used for layout, components, responsive design, and utility classes.

Bootstrap Icons were used for the social media and project icons.
